Modelo ISO/IEC 15504 (SPICE)

El estándar ISO/IEC 15504, también conocido como SPICE (Software Process Improvement and Capability Determination), es un marco para la evaluación de procesos de software y mejora de procesos.

El modelo define seis niveles de capacidad para los procesos:

graph LR A[Nivel 0: Incompleto] --> B[Nivel 1: Realizado] B --> C[Nivel 2: Gestionado] C --> D[Nivel 3: Establecido] D --> E[Nivel 4: Predecible] E --> F[Nivel 5: Optimizado]

Descripción de los niveles:

  • Nivel 0 - Incompleto: El proceso no se implementa o no alcanza sus objetivos.
  • Nivel 1 - Realizado: El proceso se implementa y alcanza sus objetivos.
  • Nivel 2 - Gestionado: El proceso se implementa de manera gestionada (planeado, monitoreado y ajustado).
  • Nivel 3 - Establecido: El proceso se implementa usando principios de ingeniería de software definidos.
  • Nivel 4 - Predecible: El proceso opera dentro de límites definidos para lograr sus objetivos.
  • Nivel 5 - Optimizado: El proceso se mejora continuamente para cumplir con los objetivos de negocio actuales y futuros.
